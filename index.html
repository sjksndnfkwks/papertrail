极客安装指南

只需 1 分钟，开启你的高阶阅读体验

  1. 准备工作：前往 ModelScope 控制台 获取您的免费 API Key。
  2. 点击下载 papertrail_1.1.zip 压缩包，并解压。
  3. 打开 Chrome，进入 chrome://extensions/ 并开启右上角 “开发者模式”。
  4. 点击 “加载已解压的扩展程序”，选择刚刚解压的文件夹。
  5. ⚠️ 必要操作：在插件详情页中，务必打开 “允许访问文件网址”，否则无法读取本地 PDF。
🚀

如何激活 AI 能力？

打开任意 PDF 后，点击侧边栏右上角的 【⚙️ 设置】 按钮。在弹窗中粘贴您在第 0 步获取的 API Key 并保存。 选择模型 （默认deepseek-ai/DeepSeek-V3.2） 即可开始您的旅程。
