极客安装指南

只需 1 分钟，开启你的高阶阅读体验

  1. 点击下载 papertrail_1.0.zip 压缩包，并解压到一个固定的文件夹。
  2. 打开 Chrome，在地址栏输入 chrome://extensions/ 并回车。
  3. 开启页面右上角的 “开发者模式” 开关。
  4. 点击左上角的 “加载已解压的扩展程序”，选择刚刚解压的文件夹。
  5. ⚠️ 极其重要：如需在本地 PDF (电脑里的文件) 上使用，请点击插件的“详细信息”，并打开 “允许访问文件网址”。

💡 首次使用提示：

打开任意 PDF 后，点击侧边栏右上角的【⚙️ 设置】按钮，填入您的 ModelScope API Key 即可起飞。（推荐选用 DeepSeek-V3.2 或 Qwen 模型）
