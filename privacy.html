Privacy Policy / 隐私权政策

This policy applies to the PaperTrail Chrome Extension. | 本政策适用于 PaperTrail 浏览器扩展。

1. Data Collection & Storage / 数据收集与存储

English: PaperTrail is designed with a "Privacy-First" architecture. We do NOT collect, store, or transmit your personal data to our own servers. All PDF text extraction and note-taking happen locally within your browser. Your API Keys and notes are stored exclusively in your browser's chrome.storage.local.

中文：PaperTrail 采用“隐私优先”架构。我们不会收集、存储或向我们的服务器传输您的任何个人数据。所有的 PDF 文本提取和笔记记录均在您的浏览器本地完成。您的 API 密钥和笔记仅保存在您浏览器的 chrome.storage.local 本地存储中。

2. Third-Party Services / 第三方服务

English: This extension interacts with ModelScope API (or other AI providers you configure) to provide summarization services. Only the text content of the current PDF page is sent to the AI provider for processing. Please refer to their respective privacy policies regarding data handling.

中文：本扩展与 ModelScope API（或您配置的其他 AI 供应商）进行交互以提供总结服务。只有当前 PDF 页面的文本内容会被发送至 AI 供应商进行处理。关于数据处理细节，请参阅对应供应商的隐私政策。

3. Permissions / 权限用途

English: We only request essential permissions (sidePanel, activeTab, storage, host permissions) to provide core features. We do not use these permissions to collect browsing history or track user behavior.

中文：我们仅申请必要的权限（侧边栏、活动标签页、存储、主机权限）以提供核心功能。我们不会利用这些权限收集浏览记录或追踪用户行为。

4. Contact / 联系方式

English: If you have any questions, please contact us via our GitHub repository.

中文：如有任何疑问，请通过我们的 GitHub 仓库联系我们。
